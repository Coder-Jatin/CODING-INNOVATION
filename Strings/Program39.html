39. Program to replace the spaces of a string with a specific character

Code


import java.util.Scanner;
public class ReplaceSpaces {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Input the string
        System.out.print("Enter a string: ");
        String inputString = scanner.nextLine();

        // Input the character to replace spaces
        System.out.print("Enter the character to replace spaces with: ");
        char replacementChar = scanner.nextLine().charAt(0);

        // Replace spaces with the specified character
        String resultString = replaceSpaces(inputString, replacementChar);

        // Output the result
        System.out.println("Original String: " + inputString);
        System.out.println("Modified String: " + resultString);
    }

    private static String replaceSpaces(String input, char replacement) {
        // Using replaceAll to replace spaces with the specified character
        return input.replaceAll(" ", String.valueOf(replacement));
    }
}

Output

Enter a string: Hello World
Enter the character to replace spaces with: -
