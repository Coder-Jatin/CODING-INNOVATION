50. Create an object StringBuffer class to show functionality of insert, append, delete and replace functions.

Code


public class StringBufferExample {
    public static void main(String[] args) {
        // Creating a StringBuffer object
        StringBuffer stringBuffer = new StringBuffer("Hello, World!");
        
        // Appending a string at the end
        stringBuffer.append(" How are you?");
        System.out.println("After Append: " + stringBuffer);
        
        // Inserting a string at a specific position
        stringBuffer.insert(7, "Awesome ");
        System.out.println("After Insert: " + stringBuffer);
        
        // Deleting characters from index 7 to 15
        stringBuffer.delete(7, 15);
        System.out.println("After Delete: " + stringBuffer);
        
       // Replacing a substring
        stringBuffer.replace(0, 5, "Hi");
        System.out.println("After Replace: " + stringBuffer);
    }
}

Output

After Append: Hello, World! How are you?
After Insert: Hello, Awesome World! How are you?
After Delete: Hello, World! How are you?
After Replace: Hi, World! How are you?
